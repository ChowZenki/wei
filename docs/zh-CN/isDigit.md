isDigit
=======

检查数据是否只由数字组成

案例
----

### 检查"abc123"是否为数字
```php
if ($widget->isDigit('abc123')) {
    echo 'Yes';
} else {
    echo 'No';
}
// 输出了'No'
```

#### 错误信息
| **名称**              | **信息**                                                       | 
|-----------------------|----------------------------------------------------------------|
| `pattern`             | %name%只能由数字(0-9)组成                                      |
| `negative`            | %name%不能只由数字(0-9)组成                                    |
| `notString`           | %name%必须是字符串                                             |
