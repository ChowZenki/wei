[attr()](http://twinh.github.com/widget/api/attr)
=================================================

Returns the value of specified key in $data

### Returns the value of specified key in $data
```php
mixed|null attr($data, $key)
```

##### 参数
* **$data** `mixed` The data, could be array or instance of \ArrayAccess
* **$key** `string` The key of data
