[isQQ()](http://twinh.github.io/widget/api/isQQ)
================================================

检查数据是否为有效的QQ号码

### 
```php
bool isQQ( $input )
```

##### 参数
* **$input** `mixed` 待验证的数据


##### 错误信息
| **名称**              | **信息**                                                       | 
|-----------------------|----------------------------------------------------------------|
| `pattern`             | %name%必须是有效的QQ号码                                       |
| `negative`            | %name%不能是QQ号码                                             |
| `notString`           | %name%必须是字符串                                             |


##### 代码范例
检查"123456"是否为有效的QQ号码
```php
<?php
 
if ($widget->isQQ('123456')) {
    echo 'Yes';
} else {
    echo 'No';
}
```
##### 运行结果
```php
'Yes'
```
