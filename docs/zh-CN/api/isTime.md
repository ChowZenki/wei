[isTime()](http://twinh.github.com/widget/api/isTime)
=====================================================

检查数据是否为合法的时间

### 
```php
bool isDate( $input [, $format ] )
```

##### 参数
* **$input** `mixed` 待验证的数据
* **$format** `string` 日期格式,默认是"H:i:s"


完整的日期格式可以查看PHP官方文档中关于[date](http://php.net/manual/zh/function.date.php)函数的格式说明.
