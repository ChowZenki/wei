Sort two-dimensional array like SQL ORDER BY clause

### Sort two-dimensional array like SQL ORDER BY clause
```php
array sort($array, $key, $type)
```

##### 参数
* **$array** `array` the two-dimensional array
* **$key** `string` the array to be sort
* **$type** `int` sort in ascending or descending order
