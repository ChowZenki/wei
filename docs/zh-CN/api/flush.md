[flush()](http://twinh.github.com/widget/api/flush)
===================================================

flush content to the browser

##### 目录
* flush($content, $sleep)

### flush content to the browser
```php
\Flush flush($content, $sleep)
```

##### 参数
* **$content** `string` the content flushes to the browser
* **$sleep** `int` the second to sleep
